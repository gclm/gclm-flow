# /gclm:ask - 智能问答

基于项目上下文回答问题。

## 用法

```
/gclm:ask <问题>
```

## 功能

1. **项目上下文问答**
   - 理解代码结构
   - 解释实现细节
   - 查找相关代码

2. **技术问题解答**
   - 最佳实践建议
   - 框架使用指南
   - 问题排查思路

3. **知识检索**
   - 查询项目文档
   - 搜索历史记忆
   - 引用相关代码

## 工作流程

1. 分析问题意图
2. 调用 `investigator` 代理收集上下文
3. 调用 `remember` 代理查询历史知识
4. 生成回答

## 输出

```markdown
## 回答

[问题答案]

### 相关代码

// 代码示例

### 参考资料
- [文档链接](path/to/doc.md)
- [历史记忆](memory://err-xxx)

### 相关问题
- [相似问题 1]
- [相似问题 2]
```

## 示例

```bash
# 询问代码实现
/gclm:ask 用户认证是如何实现的？

# 询问项目结构
/gclm:ask 这个项目的目录结构是什么？

# 询问最佳实践
/gclm:ask 在 FastAPI 中如何处理异常？

# 询问调试方法
/gclm:ask 如何调试数据库连接问题？
```

## 相关命令

- `/gclm:learn` - 记忆管理
- `/gclm:doc` - 文档管理
