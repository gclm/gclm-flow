# AGENTS.md

Gclm-Flow 全局配置，用于 Codex CLI。

## 核心理念

使用 Gclm-Flow 工作流系统进行全栈开发。

**关键原则：**
1. **文档驱动**：先读 llmdoc 建立上下文
2. **智能检测**：自动识别语言/框架
3. **统一工作流**：理解 → 规划 → 执行 → 记录
4. **记忆学习**：记录错误，提取模式
5. **安全优先**：永远不妥协安全性

| `memory` | 记忆系统 |

语言栈（自动检测）：`java-stack` `python-stack` `go-stack` `rust-stack` `frontend-stack`

## 代码风格

- 注释**非必要不形成**
- 代码自解释优于注释
- 仅做**针对性改动**
- 不使用 emoji
- 小文件优于大文件（200-400 行）
