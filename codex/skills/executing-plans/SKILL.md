# 按计划执行

按计划分批落实，不在执行中临时发明需求。

## 工作流

1. 读完整计划，先指出缺口和风险。
2. 建立任务清单，明确当前批次。
3. 逐步执行，每步都做对应验证。
4. 每个批次结束后汇报：完成项、验证证据、剩余风险。
5. 遇到阻塞立即停下，不猜。

## 适用场景

- 已经有设计/实施计划
- 任务较多，适合分批推进
- 希望在阶段边界做 review 或用户确认
